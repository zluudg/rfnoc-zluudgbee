# Introduction
Some simple blocks for GNU Radio Companion that uses RFNoC to implement parts of
the IEEE 802.15.4 standard on the FPGA in certain USRP devices.

# Installation
'''
mkdir build
cd build
cmake ..
make
sudo make install
'''
